# 梦幻西游自动估价系统

基于Excel价格表的自动估价应用，将Excel数据转换为可交互的网页估价工具。

## 已完成的文件

### 核心文件
- `auto_estimate.html` - 主估价页面（推荐使用）
- `improved_rules.js` - 改进的规则库（从Excel转换而来）
- `rules.js` - 基础规则库（从Excel转换而来）
- `test.html` - 规则测试页面

### 原始文件
- `index.html` - 原始的简易估价页面（静态规则）

### 工具脚本（位于 `../scripts/` 目录）
- `analyze_excel.py` - Excel文件结构分析工具
- `convert_excel_to_js.py` - Excel转JavaScript转换器（基础版）
- `improved_converter.py` - 改进的转换器（智能价格解析）
- `excel_to_js.py` - 原始转换器
- `excel_to_json.py` - Excel转JSON工具
- `requirements.txt` - Python依赖包列表

## 如何使用

### 1. 启动估价系统
直接双击打开 `auto_estimate.html` 文件，或在浏览器中打开该文件。

### 2. 使用步骤
1. **选择等级**：69、109、129或175级
2. **选择角色类型**：根据等级显示可用的角色选项
3. **选择限量/外观**：显示该角色的限量选项
4. **选择装备**（可选）：宝宝、无级别、特技装备
5. **计算总价**：点击"计算总价"按钮

### 3. 更新Excel数据
当Excel价格表更新时，重新运行转换脚本：

```bash
# 进入脚本目录
cd E:\gujia\scripts

# 运行改进的转换器（推荐）
py improved_converter.py

# 或者指定Excel文件路径
py improved_converter.py "E:\Users\Administrator\Desktop\新估价表.xlsx"
```

转换后刷新浏览器页面即可使用新数据。

## 价格解析逻辑

改进的转换器使用智能价格解析：
- **数字如 "18"** → 解析为18万（180,000元）
- **带"w"或"万"** → "1w" → 10,000元，"1.5万" → 15,000元
- **带"k"或"千"** → "1k" → 1,000元
- **范围如 "1000/1500"** → 取平均值1,250元
- **混合单位**：根据上下文自动判断

## 技术架构

```
Excel价格表 → Python转换脚本 → JavaScript规则库 → HTML界面
```

### 数据流
1. **Excel文件**：包含角色、宝宝、装备等价格数据
2. **转换脚本**：解析Excel，处理价格单位，生成JSON结构
3. **JavaScript规则库**：包含所有价格规则和解析函数
4. **HTML界面**：提供用户交互，调用规则库进行计算

## 注意事项

1. **价格单位**：Excel中的数字默认以"万"为单位（如18表示18万）
2. **数据更新**：修改Excel后需重新运行转换脚本
3. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Edge）
4. **数据准确性**：估价结果仅供参考，实际交易价格可能受多种因素影响

## 故障排除

### 问题：页面无法加载规则
- 检查浏览器控制台是否有错误
- 确认 `improved_rules.js` 文件存在且可访问
- 检查Excel文件路径是否正确

### 问题：价格显示不正确
- 检查Excel中的价格格式
- 运行 `analyze_excel.py` 分析数据结构
- 调整转换脚本中的价格解析逻辑

### 问题：角色选项不显示
- 检查Excel中对应等级的角色数据
- 确认转换脚本正确解析了角色工作表
