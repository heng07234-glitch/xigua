# 梦幻西游估价系统 - 首页使用说明

## 文件结构

```
mifangame-price-main/
├── index.html              # 新首页（分类导航）
├── index_backup.html       # 原估价系统备份
├── auto_estimate.html      # 灵饰估价页面（已修复）
├── home.html              # 首页副本（同index.html）
├── debug_ling_shi.html    # 灵饰数据调试页面
├── improved_rules.js      # 数据规则库
└── README_HOME.md         # 本说明文件
```

## 功能说明

### 1. 首页 (index.html)
- 包含7个估价类别：灵饰、宝宝、无级别、任务不磨、特技装备、愤怒腰带、衣服
- 目前仅**灵饰估价**功能可用，其他功能标记为"开发中"
- 响应式设计，支持手机和电脑浏览
- 点击灵饰卡片进入灵饰估价页面

### 2. 灵饰估价页面 (auto_estimate.html)
- 基于Excel"灵饰"工作表数据的估价系统
- 功能：
  - 数据表格展示所有灵饰价格
  - 多条件筛选（类型、类别、等级、属性关键词）
  - 价格详情解析（点击"查看详情"）
  - 智能价格解析（支持数字、带单位、范围等格式）
- 页面顶部有"返回首页"按钮

### 3. 数据调试页面 (debug_ling_shi.html)
- 用于检查灵饰数据结构的调试工具
- 如果灵饰页面无法加载数据，可打开此页面查看原始数据结构

### 4. 原估价系统 (index_backup.html)
- 原有的"西瓜团队估价表"备份
- 如需使用原系统，请重命名或直接打开此文件

## 使用流程

1. **打开首页**：直接打开 `index.html`
2. **使用灵饰估价**：
   - 点击"灵饰估价"卡片
   - 在灵饰页面使用筛选功能查找目标灵饰
   - 点击"查看详情"查看价格解析
   - 点击"返回首页"回到分类页面
3. **检查数据问题**：
   - 如果灵饰页面显示"加载中"或报错
   - 打开 `debug_ling_shi.html` 查看数据是否正常加载
   - 检查控制台（F12）错误信息

## 数据说明

- 数据源：`估价表.xlsx` → `improved_rules.js`
- 价格格式：
  - 数字：如 `1.7`（自动转换为1.7万=17,000元）
  - 带单位：如 `1.7w`、`3000k`
  - 范围：如 `1.7/3000`（计算平均值）
- 更新数据：修改Excel后运行 `py improved_converter.py` 重新生成 `improved_rules.js`

## 注意事项

1. 目前仅灵饰功能完整，其他类别需后续开发
2. 价格仅供参考，实际交易价格可能受市场影响
3. 如果页面显示异常，请检查浏览器控制台（F12）的错误信息
4. 原估价系统已备份为 `index_backup.html`，如需使用请自行恢复
