灵饰估价页面使用说明
=========================

一、文件说明
------------
1. 数据源: E:\Users\Administrator\Desktop\灵饰价格表.xlsx
2. 转换脚本: convert_lingshi_v2.py (Python脚本)
3. 生成的数据文件: lingshi_data.js
4. 灵饰估价页面: web\ling-shi\estimate.html
5. 批量转换脚本: convert_lingshi.bat

二、使用流程
------------
1. 更新Excel数据后，运行 convert_lingshi.bat
   - 双击 convert_lingshi.bat
   - 脚本会自动转换Excel并复制数据到前端目录

2. 打开灵饰估价页面
   - 用浏览器打开 mifangame-price-main\lingshi_estimate.html
   - 页面会自动加载最新的价格数据

三、页面功能
------------
1. 数据筛选
   - 灵饰部位: 前排、后排（后排数据暂缺）
   - 灵饰类别: 物理、法系、辅助
   - 灵饰等级: 60、80、100、120、140级
   - 装备类型: 戒指、耳饰、配饰、手镯
   - 属性关键词: 输入属性关键词搜索

2. 表格显示
   - 左侧属性/价格: 主属性组合
   - 右侧属性/价格: 副属性组合
   - 价格显示: 带千位分隔符的数字
   - 装备类型: 彩色徽标标识

四、注意事项
------------
1. 爆属性价格列目前无数据，显示为空
2. 目前只有前排灵饰数据（戒指、耳饰），后排数据（配饰、手镯）暂缺
3. 价格单位: 元（人民币）
4. 数据仅供参考，实际价格可能波动

五、技术依赖
------------
- Python 3.x
- 依赖库: openpyxl (用于读取Excel)
- 安装依赖: py -m pip install openpyxl

六、更新维护
------------
当Excel文件更新时，只需重新运行 convert_lingshi.bat 即可更新网页数据。

七、常见问题
------------
1. 转换失败: 检查Excel文件路径是否正确
2. 页面无数据: 检查浏览器控制台是否有JavaScript错误
3. 价格显示异常: 检查Excel中价格是否为数字格式
